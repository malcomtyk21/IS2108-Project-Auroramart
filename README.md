# Prerequisites
**Please ensure you have Python 3.13 or above installed in your machine**

# Setup
1. Download the ZIP folder of the project
2. Extract the contents
3. Open the `auroramart` project directory in your terminal of choice <br>

(Optional) Create a Python Virtual Environment using the following command
```terminal 
python -m venv <your-venv-name>
```

To run the virtual environment, input the following into your terminal of choice
```terminal
.\venv\Scripts\Activate
```

4. Run the following command to install all packages necessary for the project
```terminal
pip install -r requirements.txt
```
# To test
1. Navigate to the `auroramartproj` folder in your terminal
2. Run the following code in your terminal of choice
```terminal
python manage.py runserver
```
3. Open your web browser of choice and navigate to `http://localhost:8000/` to test the website features
